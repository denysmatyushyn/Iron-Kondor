# BTC Options Iron Kondor Backtest

Репозиторий содержит код для загрузки исторических опционов на BTC (Deribit), расчёта implied volatility и греков по модели Black–Scholes, а также бэктеста стратегии продажи «железного кондора». Реализовано два варианта бэктеста (со строгими и более мягкими фильтрами по данным), что позволяет сравнить влияние качества данных на количество сделок и итоговый результат.
